# Blowback — Listening Companion

Read-along notes and full transcripts for the **Blowback** podcast (hosts **Brendan James & Noah Kulwin**).

Each episode has a narration-order set of notes with `[hh:mm:ss]` jump points, real-world dates in **bold**, the hosts' analysis preserved, and the sources they cite — plus the **full transcript**.

## How to use this

- Pick a season in the left sidebar, then an episode.
- Play the episode and read along — the `[hh:mm:ss]` markers line up with the audio.
- Use the **search box** (top) to find any person, place, or event across all seasons.
